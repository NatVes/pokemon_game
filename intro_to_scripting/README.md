# Intro to scripting

## What is scripting?
- small, does the job
- purpose:
  - automate tasks
  - manipulate data
  - perform specific action that doesn't require complexity of a full fledge application

## Characteristics of scripting
- automate a repetitve tasks e.g:
  - file manipulation
  - system admin
  - batch processing
  - deployment pipeline
- Task Specific
  - to solve a particular problem
- shoter dev cycle
- intractive - will vary
- usually would be able to run from the CLI
- minimalist design

## Main differences between programming and scripting
- programming includes scripting
- scripting is usually simpler whereas programming is more complex
- scripting usually used to automate simple or routine tasks whereas programming is wider purpose
- scripting is usually execute it by an interpreter, whereas either interpreted or needs to be complied before being run.
- scripting development cycle is usally shorter whereas programming is usally longer and more invloved in SDLC phases.
- scripting learning wise is much easier.

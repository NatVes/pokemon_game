# JSON and YAML packages used

## Summary of functions used
- 'json.dumps()' - covnerts dict to json - formatted string
- 'json.dump()' - converts dict to json file
- 'json.loads()' - json-formatted string to dict
- 'json.load()' - json file to dict
- 'yaml.safe_load()' - yaml file to dict
- 'yaml.dump()' - dict to YAML file
